# Beta-oxidation-ATP-calculator
Calculates ATP yield from beta oxidation of a fatty acid given carbon chain length, saturation status, and number of double bonds if unsaturated.
input : Carbon Chain Length, saturated/unsaturated, number of double bonds if unsaturated
Assume that 1 FADH2 converts into 2 ATP after going through the electron transport chain
Results will vary depending on the P/O ratio model used.
Limitations: does not account for peroxisomal beta oxidation, assumes complete oxidation, no ketone body diversion
